== Description ==

[SendX](https://sendx.io?utm_source=wp-plugin&utm_medium=link&utm_campaign=description) is a lead-generation and marketing automation platform to grow your web business. In simple words it is marketing for non-marketers. 

Installing this plugin will provide you various widgets like hello bars, popups, sliders, modals etc. which you can use to convert your visitors to blog subscribers. Once done, you can then use SendX's drip sequences and autoresponders to send your most popular blogs, email courses, RSS feeds to your blog subscribers. 

= The SendX Process =

Simply [sign up for a forever free SendX account] (https://sendx.io/?utm_source=wp-plugin&utm_medium=link&utm_campaign=foreverfree) and install this
plugin to get started. Once installed, we place a tracking script on your wordpress blog to
activate your SendX account's features.

== Installation ==

1. Activate the **Email Marketing by SendX** plugin through the 'Plugins' menu in WordPress
2. Configure the plugin by going to the `SendX` page that appears in your Settings menu

== Frequently Asked Questions ==

= How do I install the tracking script on my blog ? =

This plugin automatically installs the SendX tracking script for you. All you 
have to do is install this plugin and enter your Team ID on its settings page.
Your Team ID can be found on the [Settings > Team Id](https://app.sendx.io/setting/?utm_source=wp-plugin&utm_medium=link&utm_campaign=setting)
page.

= Do I need to have a SendX account? =

Yep. This plugin installs the tracking code needed for SendX to work. 
[Sign up here](https://sendx.io?utm_source=wp-plugin&utm_medium=link&utm_campaign=signup) for a forever free trial.

We're always happy to help with setting it up - just drop us a mail to hello@sendx.io 
to get started!
